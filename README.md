# NotebookLM analysis for IEEE URUCON2026
Repository for the article "An Experimental Study of LLM-Assisted Learning in an Undergraduate Electrical Engineering Course", submitted to the IEEE URUCON 2026 Conference.

* Plantilla para el reporte: [Plantilla para informe - Actividad complementaria SAM 2025.pdf](https://github.com/gcapde/eval_NotebookLM_URUCON2026/blob/main/Plantilla%20para%20informe%20-%20Actividad%20complementaria%20SAM%202025.pdf).
* Lista de preguntas y videos asociados: [Lista de preguntas y videos asociados - Actividad complementaria SAM 2025.pdf](https://github.com/gcapde/eval_NotebookLM_URUCON2026/blob/main/Lista%20de%20preguntas%20y%20videos%20asociados%20-%20Actividad%20complementaria%20SAM%202025.pdf).
* Carpeta con todos los reportes para cada una de las preguntas: [reports](https://github.com/gcapde/eval_NotebookLM_URUCON2026/blob/main/reports).
